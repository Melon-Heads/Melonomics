The Software

Melonomics is designed to analyse and interpret data from transcriptomics experiments. It is optimised for the analysis of transcripts from de-novo assembled RNA-seq data. The basis of the statistical analysis is focussed around the associated FPKM values for each assembled transcript, and identifying the differentially expressed and top genes from each sample group. Melonomics outputs publication standard interactive plots that can be downloaded.

Tools

Blastn and Biopython query input transcripts sequence against a NCBI database

Publication standard interactive plots are produced using a multitude of R packages

Web interface was built using the lightweight python flask web framework
